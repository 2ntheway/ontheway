## CSS基础学习

### 基本语法

1.嵌入式样式

```css
    <style type="text/css">
        p{
            background-color: blue;
            font-size: 50px;
        }
    </style>
```

2.内联式样式

```html
<p style="color :red">这里文字是红色。</p> 
```

3.外部样式

```html
<link href="style.css" rel="stylesheet" type="text/css" />
```

选择器

1.标签选择器

```html
    <style>
```

```html
    <style>
        .p1{
            background-color:darkviolet;
        }

    </style>

<body>
    <p class="p1">今天天气阴天</p>
</body>
```
